# 江西单招

## 1. 基本概念

**单招**全称为 “高等职业教育单独招生”，考生通过文化统考和职业技能考试，按专业组分类录取。招生对象一般是：

- **中等职业学校**（如中专、职业高中）毕业生。
- 分数在300以下的普通高考生。单招录取后考生**不得再参加当年普通高考**。

## 2. 学费

- 公办院校：普通专业普遍为5000元/年，中外合作约12000-13000元/年
- 民办院校：多在9700-12000元/年，艺术类则更贵

## 3. 考试科目

- 科目1：文化统考

  - **考试形式：语文、数学合卷考试，每科100分，共200分，考试时间150分钟。**
  - 语文试卷分值：选择题$8*3$、默写$5*2$、诗歌鉴赏$13$、现代文阅读$13$、作文$40$
  - 数学试卷分值：判断题$6*3$、填空题$6*5$、填空题$4*5$、解答题$10+10+12$
- 科目2：职业适应性测试（笔试）

  - 考察语言能力、职业素养、逻辑推理能力、通用知识能力、计算机基础能力等，**总分250分**
- 科目3：职业技能测试（机试、面试等）
  - 考察职业素养与技能测试，总分250分
  - 分为：财经商贸类、旅游类、体育类、烹饪类等


> 科目1自2025年起由全省统一命题，技能考试计划在2027年改为全省统一组织

## 4. 时间表

- 2026年3月上旬：单招志愿填报
  
- 2026年3月下旬：单招考试
  
- 2026年4月：成绩发布及院校录取
